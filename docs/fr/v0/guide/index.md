# Introduction
La librairie `@duplojs/http` fait partie de l’écosystème DuploJS. Cet écosystème a pour objectif de rendre le développement d’applications en TypeScript plus robuste. `@duplojs/http` s’inscrit dans cette démarche en améliorant l’expérience de développement, afin de créer des applications plus durables, tout en restant rapide à mettre en œuvre.


## Role
Les serveurs HTTP ne sont qu’une interface permettant à un client d’échanger avec le métier. Pourtant, beaucoup de frameworks autour du protocole HTTP laissent entendre que leur rôle est de porter toute la logique métier. `@duplojs/http`, au contraire, se concentre uniquement sur sa mission : offrir une interface simple et surtout robuste pour faire transiter les données, afin que les utilisateurs comme les développeurs puissent travailler plus facilement et se consacrer à l’essentiel : le métier.
